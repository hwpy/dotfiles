### Files:
- alacritty/alactirry.toml: config for Alacritty terminal emulator
- nvim: lua config files for neovim
- ruff/pyproject.toml: ruff config
- user: config files for user dir (tmux, zsh)

### Requirements:
- npm / fnm pyright
    * npm install -g pyright
- lua
- ruff (optional)
